Let's Explore Some Maps

A collection of maps developed by James FL Thompson

Developed for
NewMapsPlus, Palmer Engineering,
and for fun!

Fayette County 2023 Crashes

A digital map showing crashes in Fayette County, KY in 2023. Map created using D3, MapLibre, and MapTiler using JavaScript, HTML and CSS.

Modernized USGS Topographic Maps

USGS topographic maps are classy and beautiful. What if we added modern elevation data to them? This album answers that question. Developed using ArcGIS Pro.

Illuminated Visibility Maps

Visibility maps for two of my favorite destinations inspired by John Nelson's online tutorials. Maps viewing Pilot Knob State Nature Sanctuary, KY and Mount San Jacinto State Park, CA. Developed using ArcGIS Pro.

Smoky Mountains Nat'l Park - 3D

The Great Smoky Mountains are a wonder to behold. This static map explores the park as a 3-Dimensional "shoebox" cutout. Developed using ArcGIS Pro.

US Power Plants

A digital proportional map created using Leaflet. This web map explores patterns of energy production. This map was created for NewMapsPlus in MAP672.

KIPDA SS4A Maps

A collection of static maps developed for Palmer Engineering in partnership with HDR Inc. for the KIPDA SS4A grant application. Maps developed using ArcGIS Pro and Python.

Kenya Boys & Girls Education Attainment

This proportional symbol map uses D3 and Leaflet to compare education rates between boys and girls in Kenya. Map developed during NewMapsPlus' MAP673 course.

US Unemployment Rates

A choropleth map showcasing unemployment rates in the US and the temporal variability from the years 2001 to 2021. Created using Leaflet for NewMapsPlus' MAP673 course.
